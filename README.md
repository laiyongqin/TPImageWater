#TPImageWater
ThinkPHP扩展的为已经上传的图片自动添加【响应式】水印。 给出标准样式的照片大小、水印大小、水印位置，循环遍历或者根据数据库字段遍历添加水印，根据比例计算合适的水印大小和位置
